## Directory structure
```bash
.
├── annotations
├── images
├── keypoints
├── pose_association.py
└── README.md
```
- Put all annotations files in Pascal VOC in annotations directory
- Put all keypoints files generated by Pose-based annotation tool
## Run
```python3
python3 pose_association.py --anno_dir annotations --keypoint_dir keypoints
```
